# Titanic Dashboard

## Problem Statement
Create an interactive dashboard using Titanic dataset to visualize passenger survival analysis dynamically.

## Dataset Details
Dataset Used: Titanic Dataset

Features:
- PassengerId
- Survived
- Pclass
- Sex
- Age
- Fare
- Embarked

## Approach
- Used Streamlit for dashboard creation
- Applied filters for gender and passenger class
- Visualized data using charts
- Created KPI metrics for analysis

## Results
- Interactive dashboard created successfully
- Survival trends visualized clearly
- Dynamic filtering implemented

## Tools Used
- Python
- Streamlit
- Pandas
- Matplotlib
- Seaborn
